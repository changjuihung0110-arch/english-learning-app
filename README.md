# 七天学英文拼音 🎓

让孩子在7天内轻松掌握26个英文字母发音和拼读的互动学习小程序！

## 功能特点 ✨

- **七天课程体系**：循序渐进的学习计划，从元音到辅音，再到单词拼读
- **互动发音**：点击字母或单词即可听到标准发音
- **趣味游戏**：字母配对、拼写挑战、趣味问答等多种游戏模式
- **进度追踪**：自动保存学习进度，记录连续学习天数
- **成就系统**：解锁徽章，激励孩子持续学习
- **本地存储**：使用localStorage保存学习数据，无需联网

## 课程安排 📚

### 第1天：元音字母基础
学习目标：掌握5个元音字母（A, E, I, O, U）的发音

### 第2天：辅音字母（一）
学习目标：掌握B, C, D, F, G的发音

### 第3天：辅音字母（二）
学习目标：掌握H, J, K, L, M的发音

### 第4天：辅音字母（三）
学习目标：掌握N, P, Q, R, S的发音

### 第5天：辅音字母（四）
学习目标：掌握T, V, W, X, Y, Z的发音

### 第6天：单词拼读实战
学习目标：运用所学字母拼读简单CVC单词

### 第7天：综合测试
学习目标：检验学习成果，颁发成就徽章

## 使用方法 🚀

### 方法一：直接打开
1. 双击 `index.html` 文件即可在浏览器中打开

### 方法二：使用本地服务器（推荐）
```bash
# 使用Python
python -m http.server 8000

# 或使用Node.js
npx http-server
```
然后在浏览器中访问 `http://localhost:8000`

## 注意事项 ⚠️

1. 首次使用需要允许浏览器使用语音功能
2. 学习数据保存在浏览器本地，清除浏览器数据会丢失进度
3. 建议使用最新版本的浏览器以获得最佳体验
4. 语音发音质量取决于浏览器和系统设置

## 常见问题 ❓

**Q：为什么听不到发音？**
A：请检查浏览器是否允许使用语音功能，并确保音量已开启。

**Q：学习进度会丢失吗？**
A：学习数据保存在浏览器本地，除非清除浏览器数据，否则不会丢失。

**Q：可以重置学习进度吗？**
A：可以在浏览器开发者工具中清除localStorage数据来重置进度。

**Q：适合多大年龄的孩子使用？**
A：适合3-8岁的孩子使用，建议家长陪同学习。
